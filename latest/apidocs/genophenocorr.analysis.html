True if the variant is recessive. Default is False.

Type:

boolean

include_unmeasured

True if we want to assume a patient without a specific phenotype listed does not have that phenotype. Otherwise, will only include those that explicitely say the phenotype was not observed. Defaults to True.

Type:

boolean

include_large_SV

True if we want to include large structural variants in the analysis. Defaults to True.

Type:

boolean

min_perc_patients_w_hpo

Will only test phenotypes that are listed in at least this percent of patients. Defaults to 10 (10%).

Type:

integer
